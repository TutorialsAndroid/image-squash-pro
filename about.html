Privacy‑first image compression, built for professionals

ImageSquash Pro was born from the frustration of trusting sensitive assets to random online compressors. We fixed that — permanently.

🔒 Why ImageSquash exists

Most image compressors upload your files to a server. That might be fine for cat photos, but not for design mockups, client deliverables, medical imagery, or confidential documents. ImageSquash is different. Every single pixel stays on your own machine — the code runs entirely in your browser using the HTML5 Canvas API. No server, no upload, no tracking.

We believe you shouldn’t have to choose between performance and privacy. Now you don’t have to.

⚡ What it does

🗜️

Extreme compression

Regularly shrinks 80 MB images to ~200 KB without noticeable quality loss. Smart resizing + modern codecs (WebP, JPEG, PNG).

📦

Batch processing

Drop dozens of files at once. Compress them all with a single click. Download as individual images or a single ZIP.

🎛️

Full control

Five presets (Maximum → Low) plus full custom mode: adjust quality, maximum dimensions, and output format.

💻

Offline capable

Once loaded, turn off your WiFi — ImageSquash works 100% offline. No network requests, ever.

🔐 Your privacy guarantee

ImageSquash never sends your files anywhere. The entire tool is a single HTML file (plus a CDN for ZIP creation). You can inspect every line of code — it’s open source and hosted on GitHub Pages.

Zero analytics, zero tracking, zero logs. We have no servers to collect your data, and no interest in doing so.

⚙️ Tech stack

  • HTML5 Canvas API
  • Vanilla JavaScript (ES6+)
  • JSZip
  • CSS3 (Grid, Flexbox, custom properties)
  • GitHub Pages
